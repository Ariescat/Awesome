搭建过程：

一开始本地搭建 gitbook 环境，build 后手动推送部署；

后来选用 gh-pages 插件进行脚本部署，但还是手动的方式；

最后采用 Github Actions，自动部署，解放双手~~

此外，还采用了 vercel 进行了网站托管。

动态语言与动态类型语言

动态语言：

动态语言或动态编程语言，Dynamic programming Language

动态语言是指程序在运行时可以改变其结构，新的函数可以被引进，已有的函数可以被删除等在结构上的变化。


动态类型语言：

动态类型语言是指在运行期间才去做数据类型检查的语言，说的是数据类型，动态语言说的是运行是改变结构，说的是代码结构。
